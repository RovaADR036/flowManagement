Shop Admin - Minimal full-stack prototype

- Frontend: React.js SPA
- Backend: Node.js + Express (REST API)
- DB: MySQL (structure ready, in-memory demo provided)

- Endpoints:
  - POST /api/login
  - GET /api/dashboard
  - GET/POST/PUT/DELETE /api/products
  - GET/POST /api/sales

- How to run (local):
  1. Start backend: node server/app.js (or npm i && npm run dev in server folder)
  2. Start frontend: install and run a dev server for the client (e.g., npm i, npm start via Vite)
